Privacy Policy for Better Officials HQ Stats

Last updated: 17/04/2026

1. Overview

Better Officials HQ Stats is a Chrome extension designed to help users view, analyse, and compare their past appointments. We take privacy seriously and are committed to handling all user data responsibly. This Privacy Policy explains what information we collect, how it is used, and how it is protected.

2. Information We Collect

We collect only the data required for the extension’s core functionality.

A. User Profile Information

We collect and process the following user-provided profile data:

  • Name
  • Profile photo
  • User ID
  • Friends list

This information is used to identify users within the extension and enable friend-based features.

B. Past Appointment Data

We collect appointment-related information such as:

  • Date and time
  • Teams and clubs
  • Division or competition
  • Location
  • Distance

This data is used to generate statistics, insights, and comparisons for the user.

3. How We Use the Information

We use collected information only for the following purposes:

  • To display user profile information
  • To retrieve, analyse, and present past appointment data
  • To enable friend requests and friend-based data access
  • To allow friends (with mutual approval) to view each other’s appointment history
  • To improve the extension’s functionality and user experience

We do not use the data for advertising, tracking, or profiling outside the extension’s features.

4. Data Transmission

Some data is transmitted to an external server to support the extension’s features. This includes:

  • User profile information
  • Past appointment data
  • Friend requests and friend approvals
  • User IDs required to establish friend connections

All transmitted data is used solely to provide the extension’s functionality.

5. Data Storage

We store only the following data:

  • User profile information (name, profile photo, user ID, friends list)
  • Past appointment data

All other data is processed in memory and discarded after use.

6. Data Visibility and Sharing

  • Names and user IDs may be visible to other users within the extension.
  • Past appointment data is locked away and not publicly visible unless two users mutually agree to become friends.
  • Once two users become friends, they may access each other’s appointment data as part of the extension’s intended functionality.
  • We do not sell, rent, or share any data with third parties.

7. Data Security

We take reasonable measures to protect stored and transmitted data from unauthorized access, alteration, or misuse. However, no system is completely secure, and we cannot guarantee absolute protection.

8. Children’s Privacy

This extension is not intended for children under 13. We do not knowingly collect personal information from children.

9. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected with a new “Last updated” date at the top of this page.
